# 复鼠表情包生成器

一个本地可运行的静态网页小工具：选择复鼠底图，输入文字，生成并下载 PNG 表情包。

## 项目结构

```text
复鼠表情包/
  index.html
  style.css
  script.js
  images/
```

## 本地预览

在项目目录运行：

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://127.0.0.1:4173
```

## 功能

- 选择 7 张复鼠底图
- 输入表情包文字
- 切换文字位置
- 下载 PNG 图片
- 复制图片到剪贴板

## 部署

这是一个纯静态网页项目，可以直接部署到 Vercel。

部署时不需要构建命令，入口文件是：

```text
index.html
```
